# TOKEN PRICE AGGREGATOR

High-performance, neo-brutalist dashboard for multi-chain token price discovery and trade analysis.

## CORE CAPABILITIES

- **Universal Aggregation**: Real-time price data from major Centralized (CEX) and Decentralized (DEX) exchanges.
- **Optimal Execution**: Automated identification of "Best Buy" and "Best Sell" venues to minimize slippage.
- **Trade Analytics**: Advanced Price Impact Calculator tailored for both Buy (USD input) and Sell (Token input) operations.
- **Security Profiling**: Integrated risk scoring and contract analysis for informed decision-making.
- **Market Visualization**: Live price charts with adjustable timeframes (24h, 7d, 30d).

## SUPPORTED ECOSYSTEMS

Full support for: Ethereum, BNB Chain, Base, Solana, Arbitrum, Polygon, Optimism, Avalanche, Fantom, and zkSync.

## SETUP & DEPLOYMENT

1. **Dependency Installation**:
   ```bash
   npm install
   ```

2. **Environment Configuration**:
   Configure necessary API keys and parameters in your `.env` file.

3. **Development Environment**:
   ```bash
   npm run dev
   ```

4. **Production Build**:
   ```bash
   npm run build
   npm start
   ```
